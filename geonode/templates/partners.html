{% trans "Data Providers and Partners" %}

{% trans "This list provides an insight in the main partners who have provided or are providing essential data." %} {% trans "For each of the organisations the main contributions for the ASDC are listed." %}

{% trans "World Meteorological Organization (WMO) and Hydrologic Research Center" %}

{% trans "Flash Flood Guidiance System (FFGS) from Hydrologic Research Center" %}

{% trans "WMO data is used for the flash flood forecasts, as well as providing snow water equivalent data used to calculate snow depth" %}

{% trans "National Oceanic and Atmospheric Administration (NOAA) - National Snow and Ice Data Center (NSIDC)" %}

{% trans "NSIDC data is used to calculate the daily snow coverage and avalanche risk. It was also used to calculate the historic snow average per month." %}

{% trans "Global Flood Monitoring System (GFMS) is supported by NOAA and is an essential component of the flood forecasting system" %}

{% trans "United States Geological Survey (USGS)" %}

{% trans "USGS Earthquake monitoring program, provides the historical earthquake data, latest epicentre data and shakemaps." %}

{% trans "Oak Ridge National Laboratory" %}

{% trans "Provided the Landscan data, which was merged with various other datasets, to estimate population numbers at village level." %}

{% trans "University Of Maryland" %}

{% trans "Global Flood Monitoring System (GFMS) is maintained by the university of Maryland. The data is essential for one component of the river flood forecasting in Afghanistan. NOAA and NASA are other partners which help create this data" %}

{% trans "National Aeronautics and Space Administration (NASA)" %}

{% trans "Global Flood Monitoring System (GFMS) is maintained by the university of Maryland. The data is essential for one component of the river flood forecasting in Afghanistan. NOAA and NASA are other partners which help create this data." %}

{% trans "NASA also provided SRTM and ASTER data." %}

{% trans "United States Agency for International Development (USAID)" %}

{% trans "The USAID MISTI project provided the Village Settlement database, one of the core datasets." %}

{% trans "Japan: Ministry of Economy, Trade and Industry (METI)" %}

{% trans "METI/J-System (Japan) and NASA provided the ASTER DEM data. This data is used to create hillshading, model avalanche areas, calculate slope and subsequently road travel time." %}

{% trans "The Advanced Spaceborne Thermal Emission and Reflection Radiometer (ASTER). The data is used to create detailed maps of land surface temperature, reflectance, and elevation." %}

{% trans "Joint Research Commission (JRC) - Global Flood Awareness System (GloFAS)" %}

{% trans "The Global Flood Awareness System (GloFAS) data will be integrated at a later stage, yet it is already beign used to assess the accuracy of the River Flood Forecast." %}

{% trans "NATO Communications and Information Agency (NCI Agency)" %}

{% trans "Afghanistan Flood Hazard Map was provided by NATO NCI Agency (previously NC3A), it is a crucial data set without which static flood risk and flash/river flood forecasting would not be possible" %}

{% trans "Further data data provided are basin layer." %}

{% trans "World Food Program (WFP)" %}

{% trans "Provided one of the initial airport datasets." %}

{% trans "Food and Agriculture Organization of the United Nations (FAO)" %}

{% trans "Land cover dataset created by John Latham of FAO, were not only crucial in determining the general land cover, but also played an essential role in estimating the population. Furthermore, FAO data is used to assess the areas at risk of natural disasters." %}

{% trans "World Health Organisation (WHO)" %}

{% trans "WHO and MoPH provided the initial health facility data, which was reviewed and adjusted. This data is essential to assess accessibility." %}

{% trans "Office for the Coordination of Humanitarian Affairs (OCHA)" %}

{% trans "Play an essential role in coordinating, relaying information and data. Both placing these online on the various OCHA maintained portals and helping organisations reach out to iMMAP." %}

{% trans "International Organization for Migration (IOM)" %}

{% trans "Has provided historical Rapid Assessment Form (RAF) information, which is used to calibrate the forecasting model." %}

{% trans "Ministry of Interior Affairs (MoIA)" %}

{% trans "Security Incident are presently mainly extracted from MoIA reports. This data is essential to assess humanitarian access. In the past there were a variety of other sources." %}

{% trans "Ministry of Public Health (MoPH)" %}

{% trans "WHO and MoPH provided the initial health facility data, which was reviewed and adjusted. This data is essential to assess accessibility." %}

{% trans "Ministry of Rural Rehabilitation and Development (MRRD)" %}

{% trans "MRRD plays an essential role in infrastructural data provision, especially segments of the rural road network." %}

AGCHO and the Independent Directorate for Local Governance (IDLG) are recognised as the sole authoritative source for legal and technical issues related to the delimitation of administrative boundaries, composition, and division of the administrative units of Afghanistan." %}
